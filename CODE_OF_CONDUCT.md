# Code of Conduct

This project follows a simple standard: be respectful, constructive, and careful with other people's time and data.

## Expected behavior

- Be welcoming and professional
- Focus criticism on code and behavior, not people
- Share context and rationale generously
- Treat data-governance concerns seriously

## Unacceptable behavior

- Harassment or personal attacks
- Publishing private study data or credentials
- Dismissing good-faith questions from new contributors
